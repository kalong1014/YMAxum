# YMAxum 核心层技术文档

## 1. 概述
YMAxum 核心层是整个框架的基础，包含了框架的核心功能和基础组件。核心层采用 Rust 语言开发，实现了高性能、安全、可扩展的架构设计。

## 2. 目录结构
```
src/core/
├── auth_middleware.rs     # 认证中间件
├── cache.rs              # 缓存管理
├── context.rs            # 上下文管理
├── cross_platform.rs     # 跨平台支持
├── database.rs           # 数据库管理
├── deps.rs               # 依赖管理
├── error_recovery.rs     # 错误恢复
├── http_3.rs             # HTTP/3 支持
├── iterate_api.rs        # 迭代 API
├── lib.rs                # 核心库入口
├── middleware.rs         # 中间件管理
├── mod.rs                # 模块定义
├── plugin_route_manager.rs # 插件路由管理
├── route.rs              # 路由管理
├── route_generator.rs    # 路由生成
├── state.rs              # 状态管理
└── ...
```

## 3. 核心功能

### 3.1 状态管理
- **功能**：实现全局状态的管理和共享
- **实现**：使用 `Arc<Mutex<>>` 包装核心对象，确保线程安全
- **创新点**：
  - 基于 Rust 的所有权模型，实现安全的状态共享
  - 支持状态的动态更新和实时同步
  - 提供状态变更的事件通知机制

### 3.2 路由管理
- **功能**：处理 HTTP 请求的路由和分发
- **实现**：基于 axum 框架的路由系统，实现自定义路由逻辑
- **创新点**：
  - 动态路由生成，支持插件的路由注册
  - 路由优先级管理，解决路由冲突
  - 路由参数的智能解析和验证

### 3.3 中间件系统
- **功能**：提供请求处理的中间件机制
- **实现**：自定义中间件链，支持多个中间件的组合
- **创新点**：
  - 中间件的动态注册和排序
  - 中间件的条件执行，基于请求路径和方法
  - 中间件的错误处理和恢复机制

### 3.4 跨平台支持
- **功能**：实现跨平台的系统调用和功能适配
- **实现**：使用条件编译和平台特定代码，适配不同操作系统
- **创新点**：
  - 统一的跨平台 API，屏蔽平台差异
  - 平台特定功能的优雅降级
  - 跨平台的路径处理和系统资源管理

### 3.5 错误处理
- **功能**：统一的错误处理和恢复机制
- **实现**：基于 `thiserror` 和 `anyhow` 库，实现层次化的错误处理
- **创新点**：
  - 错误的分类和标准化
  - 错误的链式传递和上下文保留
  - 错误的自动转换和用户友好的错误信息

## 4. 技术架构

### 4.1 整体架构
核心层采用模块化设计，各模块之间通过清晰的接口进行通信。整体架构如下：

```
核心层架构
├── 状态管理模块
├── 路由管理模块
├── 中间件系统
├── 跨平台支持
├── 错误处理系统
└── 工具函数库
```

### 4.2 数据流
1. 请求进入 -> 路由匹配 -> 中间件处理 -> 业务逻辑 -> 响应返回
2. 状态变更 -> 事件通知 -> 状态同步 -> 持久化存储

## 5. 创新点说明

### 5.1 技术创新
1. **高性能架构**：基于 Rust 的内存安全和零成本抽象，实现高性能的请求处理
2. **模块化设计**：核心功能与插件系统分离，便于维护和扩展
3. **安全机制**：多层安全防护，包括参数验证、权限控制、数据加密等
4. **可扩展性**：插件系统支持热插拔，无需重启服务即可更新功能
5. **跨平台适配**：统一的跨平台 API，支持 Windows、macOS、Linux 等多个平台

### 5.2 业务创新
1. **灵活的路由系统**：支持动态路由和插件路由注册，适应复杂的业务场景
2. **智能中间件**：条件执行和优先级管理，满足不同业务需求
3. **统一的状态管理**：全局状态的安全共享和实时同步，简化业务逻辑
4. **优雅的错误处理**：层次化的错误处理和用户友好的错误信息，提升用户体验
5. **高效的依赖管理**：智能的依赖解析和版本控制，确保系统稳定性
