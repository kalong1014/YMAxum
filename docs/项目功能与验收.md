# 项目功能与验收

## 一、GUF集成架构

### 1. 整体架构
- **架构风格**：分层架构，采用适配器模式
- **核心层次**：
  - 应用层：业务逻辑和用户界面
  - 集成层：GUF适配器和组件管理
  - 核心层：核心功能和插件系统
  - 基础设施层：数据库、缓存、安全等

### 2. 核心组件

| 组件名称 | 功能描述 | 所属模块 | 交互方式 |
|----------|----------|----------|----------|
| GUF适配器 | 处理与GUF框架的通信 | guf/adapter | WebSocket/HTTP |
| 组件管理器 | 管理GUF组件生命周期 | guf/component_manager | 事件驱动 |
| 配置同步器 | 实现配置双向同步 | guf/config_sync | 定期同步/事件触发 |
| 事件总线 | 处理事件发布和订阅 | guf/event_bus | 发布-订阅模式 |
| 插件系统 | 管理GUF插件 | guf/plugin_system | 插件API |

### 3. 组件交互方式
- **适配器与GUF框架**：
  - WebSocket长连接：实时通信
  - HTTP API：配置同步和组件操作
- **组件管理器与适配器**：
  - 事件通知：组件状态变更
  - 方法调用：组件操作请求
- **配置同步器与其他组件**：
  - 配置更新事件：通知配置变更
  - 配置查询接口：获取当前配置
- **事件总线与所有组件**：
  - 事件发布：组件状态变更、用户操作等
  - 事件订阅：监听相关事件
- **插件系统与其他组件**：
  - 插件注册：注册新插件
  - 插件调用：调用插件功能

## 五、玩家快速上手指南

### 1. 环境准备
- **系统要求**：
  - Windows 10/11 64位
  - 8GB以上内存
  - 50GB以上硬盘空间
  - .NET Framework 4.8或更高版本
- **软件安装**：
  1. 下载最新版本的YMAxum安装包
  2. 双击安装包，按照向导完成安装
  3. 启动YMAxum应用程序

### 2. 首次启动
- **配置向导**：
  1. 选择语言（支持中文、英文）
  2. 设置管理员账号和密码
  3. 配置数据库连接（可选，默认使用内置SQLite）
  4. 选择初始场景（根据行业需求）
- **初始化完成**：
  - 系统自动创建初始数据
  - 启动完成后进入登录界面

### 3. 登录系统
- **登录步骤**：
  1. 打开YMAxum应用程序
  2. 输入管理员账号和密码
  3. 点击"登录"按钮
- **登录成功**：
  - 进入系统控制台
  - 显示系统状态和快捷操作

### 4. 核心功能使用

#### 插件管理
1. **插件市场**：
   - 点击左侧菜单"插件管理" → "插件市场"
   - 浏览可用插件，点击"安装"按钮
   - 等待安装完成，点击"启用"按钮

2. **本地插件**：
   - 点击左侧菜单"插件管理" → "本地插件"
   - 查看已安装插件状态
   - 点击"启动"/"停止"按钮管理插件

#### 场景适配
1. **场景切换**：
   - 点击顶部导航栏"场景"下拉菜单
   - 选择需要的行业场景
   - 系统自动切换到对应场景

2. **场景配置**：
   - 点击左侧菜单"场景管理" → "场景配置"
   - 根据业务需求调整场景参数
   - 点击"保存"按钮应用配置

#### GUF界面
1. **界面编辑器**：
   - 点击左侧菜单"GUF" → "界面编辑器"
   - 拖拽组件创建界面
   - 配置组件属性和事件

2. **界面管理**：
   - 点击左侧菜单"GUF" → "界面管理"
   - 查看和管理已创建的界面
   - 点击"预览"按钮查看界面效果

### 5. 系统管理

#### 系统设置
1. **基本设置**：
   - 点击左侧菜单"系统管理" → "基本设置"
   - 配置系统名称、Logo、时区等
   - 点击"保存"按钮应用设置

2. **安全设置**：
   - 点击左侧菜单"系统管理" → "安全设置"
   - 配置认证方式、密码策略等

1. **系统监控**：
   - 点击左侧菜单"监控管理" → "系统监控"

2. **日志管理**：
   - 点击左侧菜单"监控管理" → "日志管理"
   - 查看系统日志、错误日志等
   - 配置日志级别和存储策略

### 6. 常见问题

#### 登录问题
- **忘记密码**：
  1. 点击登录界面"忘记密码"链接
  2. 输入管理员邮箱
  3. 按照邮件提示重置密码

- **登录失败**：
  1. 检查账号密码是否正确
  2. 检查网络连接是否正常
  3. 检查系统服务是否运行

#### 插件问题
- **插件安装失败**：
  1. 检查网络连接是否正常
  2. 检查插件签名是否有效
  3. 检查系统版本是否兼容

- **插件运行异常**：
  1. 查看插件日志
  2. 检查插件配置是否正确
  3. 尝试重启插件

#### GUF问题
- **界面加载失败**：
  1. 检查网络连接是否正常
  2. 检查GUF服务是否运行
  3. 清除浏览器缓存

- **组件操作失败**：
  1. 检查组件权限是否正确
  2. 检查组件配置是否有效
  3. 查看GUF日志
